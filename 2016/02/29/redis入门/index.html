redis入门

前阵子拜读了《redis入门指南第2版》，书中对redis的介绍通俗易懂，以下为简单的思维导图：

思维导图

由于图片过小可能看不清楚，以下为整本书的简单框架：

整体框架

redis

  1. 准备
    1. 安装——brew install redis
    2. 启动服务端
      1. redis-server [–port xxx]
      2. 初始化脚本自启动
    3. 启动客户端——redis-cli
    4. 测试连通性——在客户端中ping
    5. 停止服务端——在客户端shutdown
  2. 数据结构（字典结构）
    1. 字符串类型：序列化和反序列化
    2. 散列类型：键-字段-字段值（字符串类型）
    3. 列表类型：双向链表
    4. 集合类型——支持集合操作
    5. 有序集合类型——跳跃表（Skip list）
  3. 进阶
    1. 事务——MULTI+EXEC
    2. 监视——WATCH
    3. 过期时间——EXPIRE
    4. 排序
      1. SORT
      2. BY
      3. GET
      4. STORE
    5. 消息通知
      1. 任务队列——优先级队列
      2. 发布/订阅
    6. 管道
    7. 节省空间——编码方式
  4. 实践
    1. php——用户注册登录
    2. ruby——自动补全
    3. python——查看在线好友
    4. node.js——ip地址查询
    5. 其他语言
  5. 脚本——Lua
  6. 持久化
    1. RDB方式——使用“快照”，将内存中的所有数据生成一份副本并存储在硬盘上
      1. AOF方式——记录redis的每一条写命令
  7. 集群
    1. 复制
      1. 主从复制
      2. 读写分离
      3. 无硬盘复制
      4. 增量复制
        1. 哨兵
        2. 集群
  8. 管理
    1. 安全
    2. 通信协议
      1. 简单协议
      2. 统一请求协议
    3. 管理工具
      1. phpRedisAdmin
      2. Rdbtools

进阶学习

由于对redis数据库的应用并没有具体情景，接下来可以对redis进行源码研读，其中有黄建宏大神的《redis设计与实现》，而且他在github上面还有针对redis源码的中文注释，传送门在此——黄建宏：redis设计与实现，马克之！
